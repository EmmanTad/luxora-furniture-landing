# Luxora Furniture Landing Page

A premium furniture landing page built with pure HTML, CSS.

## Features
- Modern hero section
- Horizontal product carousel
- Parallax background
- Fully responsive
- No dependencies

## Usage
Open `index.html` in a browser or deploy via GitHub Pages.

Works on:
- GitHub Pages

- Vercel
